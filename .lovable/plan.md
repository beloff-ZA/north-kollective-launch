

# North Kollective Studio — Coming Soon Website

A premium, ultra-minimal single-page "Coming Soon" site with dark-mode-first aesthetic, Swiss-inspired layout, and sophisticated animations.

---

## 1. Design System Setup
- **Color palette**: Deep charcoal background, off-white text, warm grey accents, muted champagne/brushed gold for subtle highlights
- **Typography**: Modern sans-serif pairing with elegant letter-spacing and generous line heights
- **Dark-mode default** aesthetic throughout

## 2. Hero Section (Full Screen)
- Large, spaced-out logo text: **NORTH KOLLECTIVE STUDIO**
- Tagline: *"Beyond Expected Standards."*
- Supporting subtext about boutique marketing and creative strategy
- Subtle animated background (slow gradient shift with soft grain texture overlay)
- Smooth fade-in entrance animation on page load
- Minimal "Launching Soon" pill button

## 3. About Preview Section
- Clean two-column layout: text on one side, abstract architectural placeholder image on the other
- Short paragraph about the founder's vision, design-led thinking, and strategic creativity
- Generous whitespace, elegant typography hierarchy

## 4. Services Preview Section
- Three minimal service blocks: **Brand Strategy**, **Creative Direction**, **Digital Presence**
- Each with a short placeholder description
- Subtle hover animations (soft scale or underline reveal)
- Clean grid layout with balanced spacing

## 5. Contact / Notify Section
- Heading: *"Be first to know."*
- Email input field with minimal styling
- Submit button in champagne/gold accent
- Social icon placeholders for LinkedIn and Instagram

## 6. Footer
- © 2026 North Kollective Studio
- Location: South Africa
- Minimal, single-line layout

## 7. Animations & Polish
- Scroll-triggered fade + slide-up transitions for each section
- Micro hover effects on interactive elements
- Performance-optimized CSS animations
- Fully responsive across all breakpoints
